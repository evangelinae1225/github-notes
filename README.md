# github-notes

## Definitions 

* **Git** - is a Version Control Software made by the same person who created Linux. 

* **GitHub** - GitHub is a web service where you can sign up and have an account, and use git on a server and/ or on a website.
  
* **Repositary** - also known as "repo", is a project filled with files.
  
* **Commit** - is another term for saving.
  * Each Commit has a Hash 
  * It is possible to name each commit
  * View every Commit in history
  * +/Green Shows what was added
  * -/Red shows what has been deleted
